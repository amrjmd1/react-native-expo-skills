# Mobile Observability Reliability Plugin

Operations pack for production telemetry, alerting, and incident response.

## Scope

Use this pack for log/event standards, crash analytics enrichment, SLO-aligned alerting, and runbook-driven incident handling.

## Skills

1. `mobile-observability-telemetry`
Structured telemetry model for logs, crash events, analytics, and privacy-safe instrumentation.

2. `mobile-alerting-incident-response`
Alert policy, escalation, rollback decisions, and postmortem workflows.

## Typical Senior Use Cases

- Reduce blind spots in production diagnostics.
- Cut MTTR with standardized triage and response playbooks.
- Improve signal-to-noise ratio in mobile alerting.

## Quality Bar

- Actionable telemetry taxonomy.
- SLO/error-budget aware alerting.
- Rehearsable incident and rollback procedures.
