# Mobile Delivery Lifecycle Plugin

Operational pack for CI/CD maturity and framework migration safety.

## Scope

Use this pack for build pipeline architecture, quality gates, release governance, and RN/Expo upgrade planning.

## Skills

1. `mobile-cicd-workflows`
Deterministic CI/CD pipelines, gate policy, secret/signing governance, rollback controls.

2. `mobile-migration-upgrades`
RN/Expo major upgrade compatibility mapping, staged execution, risk-managed rollout.

## Typical Senior Use Cases

- Reduce deployment risk with branch/profile-specific release controls.
- Design automated build/test gates that catch regressions early.
- Execute major framework upgrades without destabilizing production.

## Quality Bar

- Repeatable delivery with traceable release decisions.
- Explicit compatibility and rollback strategy.
- Measurable post-upgrade validation standards.
