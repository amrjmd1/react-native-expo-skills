# Mobile Platform Orchestrator Plugin

Single-entry plugin for teams that want one skill to handle broad React Native/Expo needs and route work into specialized domains.

## Skill Included

- `mobile-platform-orchestrator`

## Why This Plugin Is Strong

- Fast onboarding: developers start with one clear entry point.
- Senior triage: request classification + domain routing + risk framing.
- Scalable workflow: keeps specialist depth available when needed.

## Recommended Usage

1. Use orchestrator first for broad/ambiguous tasks.
2. Follow routed specialist domains for deep implementation.
3. For critical production tasks, install specialist packs alongside orchestrator.

## Installation Note

For best results, install this plugin with specialist packs so orchestration can map directly to deep domain skills.
