# Mobile Revenue Systems Plugin

Revenue-critical pack for purchases, subscriptions, and entitlement integrity.

## Scope

Use this pack for purchase flow design, entitlement state machines, billing failure recovery, and subscription lifecycle controls.

## Skills

1. `mobile-payments-subscriptions`
End-to-end payments/subscription system design with consistency and failure recovery.

## Typical Senior Use Cases

- Prevent entitlement drift between client/server/store state.
- Handle renewals, grace periods, and failed billing with clear UX.
- Build auditable purchase flows with robust reconciliation.

## Quality Bar

- Deterministic entitlement ownership.
- Strong failure and abuse handling.
- Compliance-aware purchase lifecycle design.
