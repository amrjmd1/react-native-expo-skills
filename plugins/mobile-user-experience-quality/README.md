# Mobile User Experience Quality Plugin

Quality pack for accessibility, localization, and background interaction systems.

## Scope

Use this pack for a11y compliance, i18n/l10n correctness, background processing, and notification reliability.

## Skills

1. `mobile-accessibility-i18n`
Accessibility semantics, localization architecture, RTL/dynamic type resilience.

2. `mobile-background-notifications`
Background task models, push lifecycle, token handling, and delivery reliability.

## Typical Senior Use Cases

- Fix accessibility and localization regressions in complex flows.
- Design robust notification systems across iOS/Android constraints.
- Validate background behavior under permission and app-state variability.

## Quality Bar

- Inclusive-by-default UX.
- Localization-safe component patterns.
- Reliable background and notification behavior under real-device conditions.
